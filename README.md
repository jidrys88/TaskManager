TaskManager App

Beschreibung:
TaskManager ist eine Desktop-Anwendung mit WPF Frontend und ASP.NET Core Minimal API Backend (.NET 8).
Benutzer können Tasks erstellen, bearbeiten, löschen und für jede Task eigene Aktionen / Trigger definieren.

Architektur

Frontend: WPF (MVVM)

Backend: ASP.NET Core Minimal API (.NET 8)

Datenbank: SQLite

Kommunikation: HTTP/JSON

Features

Tasks erstellen, bearbeiten, löschen

Status: offen / erledigt

Trigger für jede Task (z. B. Deadline überschritten)

Aktionen pro Task (UI-Reaktion, API-Update)

Swagger/OpenAPI für einfache API-Tests
